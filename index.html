Jexactyl.

A fast, intuitive and powerful fork of Pterodactyl®.

Speed².

Jexactyl is built with speed and power in mind. A clean, fluent Laravel backend supports the React user interface with organised API endpoints.

Secure.

Jexactyl cracks down on security with features such as security key support - for fingerprints, YubiKeys and more. All in one convenient package.

Up-to-date.

Jexactyl's dependencies are constantly updated and improved to improve speed and security. Gone are the days of outdated and deprecated packages.

We're on a mission to create the best game panel available.

Notifications and Logging

Jexactyl automagically logs and audits activity on servers and accounts without exposing or storing any sensetive data. As a user, you can choose to delete as much or as little of your logs as you'd like. As for admins - only server logs are recorded and viewable to ensure privacy on a per-user basis.

Dark React UI

While Pterodactyl® exposes a beautiful and fluent UI for users, we've turned it up a notch by tinting the interface components to make them more visually-appealing. Using React provides the software with a new and fast SPA which includes a fast API and as-needed rendering.

Customisable Store System

One of the main features of Jexactyl is the powerful yet elegant store system. Users can be assigned credits (either by administrators or purchasing them through PayPal) which they can then use to deploy servers on the platform. Costs and payment gateways can be fully configured by administrators for ease of use.
